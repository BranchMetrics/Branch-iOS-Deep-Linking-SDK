The Planets Webview Example

The Planets Webview example shows how Branch links can be used in apps that have web views.

This page is a web view, and this link to the planet Saturn links back to content in the app.

The example also shows how to share Branch links that point to web view app content. The content view of each planet can be shared by email, etc., and those links will link to the content in the app, if the user has the app, or to content on the web.
